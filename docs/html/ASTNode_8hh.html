Detailed Description

Abstract Syntax Tree node for constraint expressions.

This file defines the ASTNode class which represents constraint expressions as abstract syntax trees. It supports conversion to CNF (Conjunctive Normal Form) using either Tseitin transformation or direct conversion methods.
